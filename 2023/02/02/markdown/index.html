一、Markdown介绍

1.Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。
2.它使用易读易写的纯文本格式编写文档
3.可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。
4.因简洁、高效、易读、易写，Markdown被大量使用，如Github、Wikipedia、简书等。

二、Markdown作用

1.网站：Markdown 是为 web 而设计的，是做笔记、为网站创建内容以及生成可打印文档的快速、简便的方法
2.文件资料：Markdown 文档创作工具来创建 Markdown 格式的文档并将其导出为 PDF 或 HTML 格式
等等

九、Markdown 内嵌 HTML 标签

1、说明

对于 Markdown 涵盖范围之外的标签，都可以直接在文件里面使用 HTML 本身。 如需使用 HTML，不需要额外标注这是 HTML 或是 Markdown，只需 将HTML 标签添加到 Markdown 文本中即可。

2、了解一下HTML元素特点

元素模式 元素排列 设置样式 默认宽度 包含
行内元素 一行可以放多个行内元素 不可以设置宽高 本身内容的宽高 容纳其他行内元素及文本
块级元素 一行只能放一个块级元素 可以设置宽高 容器的100% 可以包含任何标签
行内块元素 一行可以放多个行内块元素 可以设置宽高 本身内容的宽度 可以包含任何标签

3、一些规则

1.  HTML的行内元素 :  a,b,u,span,img,input,strong,label,em,h1~h6等
不受限制，都可以在 Markdown 的段落、列表或是标题里任意使用
2.  HTML的块级元素 : div table pre p 必须在前后加上空行，以便于内容区分
而且这些元素的开始与结尾标签，不可以用 tab 或是空白来缩进
3. HTML 标签中，markdown语法会失效
